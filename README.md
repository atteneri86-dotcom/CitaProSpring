# CitaPro - Gestión de Clientes

## Descripción
Aplicación web desarrollada con Spring Boot para la gestión de clientes. Permite crear, consultar, actualizar y eliminar clientes (CRUD completo).

## Tecnologías utilizadas
- Java 17
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
- JUnit

## Funcionalidades
- Crear clientes
- Listar clientes
- Editar clientes
- Eliminar clientes
- Validación de formularios
- Confirmación antes de borrar

## Base de datos
Se utiliza MySQL con una base de datos llamada `citapro`.

## Tests
Se han implementado pruebas unitarias del repositorio utilizando JUnit.

## Control de versiones
Proyecto gestionado con Git y GitHub.

## Metodología
Se ha seguido metodología Kanban utilizando Trello.
